# GMAO Intelligente – Backend

API REST Spring Boot pour l'application GMAO de Suprajit Maroc.

## Stack
- Spring Boot 3 · Spring Security · JWT · Spring Data JPA
- MySQL · Groq API · iText 7 · Socket.IO · Docker

## Démarrage rapide
```bash
# Prérequis : Java 17, Maven, MySQL (XAMPP)
mvn spring-boot:run
```

## Documentation API
Swagger UI : http://localhost:8080/swagger-ui.html
